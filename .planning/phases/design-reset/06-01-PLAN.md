# Plan 06-01: Fix Broken Imports

## Objective
Find and fix all broken imports resulting from deleted files.

## Context
After deleting animation/style files, many components will have broken imports. This plan identifies and fixes them.

## Tasks

### Task 1: Search for animation imports
- **Type**: search-and-fix
- **Pattern**: Search for imports containing:
  - `from '@/lib/gsap'`
  - `from '@/lib/animation'`
  - `from '@/lib/animations'`
  - `from '@/components/animation'`
  - `from '@/hooks/useAnimations'`
- **Action**: Remove entire import statements and any usage of imported functions
- **Done**: [ ]

### Task 2: Search for deleted component imports
- **Type**: search-and-fix
- **Pattern**: Search for imports containing:
  - `AnimatedButton`
  - `AnimatedCard`
  - `AnimatedIcon`
  - `animated-card`
  - `animated-container`
  - `headline-outside`
  - `HeadlineOutside`
- **Action**: Replace with standard shadcn alternatives:
  - `AnimatedButton` → `Button`
  - `AnimatedCard` → `Card`

- **Done**: [ ]

### Task 3: Search for deleted variant imports
