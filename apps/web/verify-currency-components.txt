## Required npm Packages

To install missing dependencies, run:

```bash
npm install cmdk @radix-ui/react-tooltip @radix-ui/react-dialog
```

## Environment Variables

Add to .env.local:

```env
NEXT_PUBLIC_API_URL=http://localhost:3000/api
```

## Quick Test

To test the components:

1. Install dependencies
2. Import demo component in a page:

```tsx
import { CurrencyDemo } from '@/components/currency/CurrencyDemo';

export default function TestPage() {
  return <CurrencyDemo />;
}
```

3. Navigate to the page and verify all components render
