- Max width: 1280px centered

- Quick nav: Single row
- Disclaimers: 2 columns

### Tablet (768px - 1023px)
- Max width: 100% with padding
- Category cards: Full width
- Quick nav: Wrap to 2 rows
- Disclaimers: Single column

### Mobile (< 768px)
- Full width with 16px padding
- Category cards: Stack vertically
- Quick nav: Horizontal scroll
- Hero text: Smaller (text-4xl)
- Buttons: Stack vertically
