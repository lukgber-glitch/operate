## User Flow

### Flow 1: Standard PDF Download

1. User clicks "Download" button
2. Dialog opens with Standard PDF selected by default
3. User clicks "Download"
4. File downloads as standard PDF

### Flow 2: ZUGFeRD Download

1. User clicks "Download" button
2. Dialog opens
3. User selects "ZUGFeRD" from format dropdown
4. ZUGFeRD Profile selector appears (animated fade-in)
5. User selects profile (EN16931 is default)
6. User clicks "Download"
7. File downloads as ZUGFeRD PDF with embedded XML

### Flow 3: XRechnung Download

1. User clicks "Download" button
2. Dialog opens
3. User selects "XRechnung" from format dropdown
